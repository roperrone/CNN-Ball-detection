# Starting instructions

Use python3 !

1. Clone this project
2. Create directory data/
3. Download [dataset files](https://perso.liris.cnrs.fr/christian.wolf/teaching/deeplearning/data/train.tgz) to data/ and unzip
4. Run python split_files.py to create train and val datasets
5. Create training loop etc.
